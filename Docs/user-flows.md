# 📱 PesaScope — Features & User Stories

## 1. 🔐 Onboarding & Permissions

**1.1** As a new user, I want a simple welcome screen that explains what PesaScope does so I know what to expect

**1.2** As a user, I want to be prompted to grant SMS read permission with a clear explanation of why it's needed

**1.3** As a user, if I deny SMS permission, I want to see a helpful message explaining the app can't work without it and a button to retry

**1.4** As a user, after granting permission, I want the app to automatically import all my existing M-Pesa messages on first launch

**1.5** As a user, I want to see a progress indicator while my messages are being imported

---

## 2. 📊 Dashboard

**2.1** As a user, I want to see my total money in, money out, and net balance for the current month at a glance

**2.2** As a user, I want to see a bar chart of my daily or weekly spending for the current month

**2.3** As a user, I want to see my top 3 spending categories this month so I know where most of my money goes

**2.4** As a user, I want to see my most recent 5 transactions on the dashboard for a quick overview

**2.5** As a user, I want to switch the dashboard between current month, last month, and a custom date range

---

## 3. 📋 Transactions

**3.1** As a user, I want to see a full chronological list of all my M-Pesa transactions

**3.2** As a user, I want each transaction to show the amount, type, counterparty name, category, and date clearly

**3.3** As a user, I want to search transactions by name, amount, or transaction code

**3.4** As a user, I want to filter transactions by type (sent, received, withdrawal, paybill, buy goods, airtime)

**3.5** As a user, I want to filter transactions by date range

**3.6** As a user, I want to filter transactions by category

**3.7** As a user, I want to tap a transaction to see its full details including the original SMS message

**3.8** As a user, I want to manually edit the category of a transaction in case it was miscategorized

**3.9** As a user, I want to add a personal note to any transaction (e.g. "rent for June")

---

## 4. 🏷️ Categories

**4.1** As a user, I want to see a donut or pie chart breaking down my spending by category for the current month

**4.2** As a user, I want to tap a category slice to see all transactions in that category

**4.3** As a user, I want to see a list of all categories with their total spend and percentage of overall spending

**4.4** As a user, I want to create a custom category with a name and icon/color

**4.5** As a user, I want to edit or delete a custom category

**4.6** As a user, I want to set auto-categorization rules — e.g. "always categorize paybill 888880 as Utilities"

---

## 5. 💰 Budgets

**5.1** As a user, I want to set a monthly spending limit for each category

**5.2** As a user, I want to see a progress bar per category showing how much of my budget I've used

**5.3** As a user, I want to be notified when I reach 80% of a category budget

**5.4** As a user, I want to be notified when I exceed a category budget

**5.5** As a user, I want to set an overall monthly spending budget across all categories

**5.6** As a user, I want budgets to automatically reset at the start of each month

**5.7** As a user, I want to see how my current spending compares to the same period last month

---

## 6. 🔄 Sync & Data

**6.1** As a user, I want the app to automatically check for new M-Pesa messages every time I open it

**6.2** As a user, I want to manually trigger a sync from the settings page

**6.3** As a user, I want the app to avoid importing duplicate transactions

**6.4** As a user, I want to see when the last sync happened

---

## 7. 📤 Export & Reports

**7.1** As a user, I want to export my transactions as a CSV file for a selected date range

**7.2** As a user, I want to share a monthly spending summary as an image or PDF

---

## 8. ⚙️ Settings

**8.1** As a user, I want to switch between light and dark mode

**8.2** As a user, I want to set my currency display preference (Ksh vs KES)

**8.3** As a user, I want to enable or disable budget notifications

**8.4** As a user, I want to clear all app data and start fresh

**8.5** As a user, I want to see the app version and a link to send feedback

---

## 9. 🔒 Security

**9.1** As a user, I want to lock the app with a PIN or biometric (fingerprint) so others can't see my finances

**9.2** As a user, I want to set up app lock during onboarding and be able to disable it in settings
